Anikit

Animation Toolkit for CSS Animation, WebGL and JavaScript

About

Anikit ( stands for Animation toolkit ) is an utility for generating CSS Animation from JavaScript functions, with a set of prebuilt CSS animations. Following is an example powered by anikit:

Concept

Anikit works with animations written as ldEditor modules. Every module contains a set of configurations calls "edits". Given a set of values for each edit, anikit generates a CSS keyframes for it.

Some modules ship along with set of edit values called "preset". Every preset ships with a name, and Anikit build named CSS animation and corresponding CSS classe for each preset, bundling it into anikit.bundle.css.

  • module - define the rules, configs about how to generate an animation.
  • edit - configuration of a module
  • preset - a set of configuration values for a module

Library

Anikit presets are separated into two groups by their purpose and released under MIT License.

loading.css

CSS library dedicated for seamless repeatable animations. A set of pure css shapes are also included, for quickly setup a css spinner with animation preset.

transition.css

CSS library dedicated for transitional animations ( both entering and exiting transitions ).

UI Kits

Anikit ships along with a Pug UI kits for easily building an interface of choosing animation.

Build Your Own Kit

If none of above fits your need, you can build your own kit with Pug variables provided in anikit.pug from scratch. Following is an example:

div each g,$index in anikit.groupName div= g each m in anikit.members[$index] div= m

Above example loops over every animation in each group, and render their name in a div tag.

APIs

While Anikit can be used to generate CSS animation based on JavaScript functions, you can also use it to animate thing directly via JS.

First, you will have to create an anikit object. It's quite straight forward to do some based on presets - just create an object with the desired preset name:

kit = new anikit("breath");

To animate an element, use animateJs method with a given time t between 0 and 1; this will update the element's style automatically:

kit.animteJs(node, t);

This design makes it possible to animate things beyond HTML and CSS with the same code, for example, WebGL shapes. Anikit provides a handy function animateThree to animate THREE.js powered meshes:

kit.animateThree(node, t);

Reference

Here we list all APIs available in the anikit object:

  • animate(node, opt) - animate node with CSS animation, with optional additional config in opt.
  • statify(node)- remove animation from node.
  • getdom()- get related style dom element for the injected CSS animation.
  • destroy()- remove the related DOM element.
  • setConfig(configObj)- set values for edits.
  • getConfig- get current values for edits.
  • origin(node,host,opt)- setup transform origin for node.
    • node - node we want to update.
    • host - host element. can be an element, or a bounding box for reference.
    • opt - options with following possible members:
      • x - origin in x coordinate. between 0 to 1; 0.5 means center of node.
      • y - origin in y coordinate. between 0 to 1; 0.5 means center of node.
      • ox, oy, s - possibly deprecated. control offset in x,y and scale when host is an element.
